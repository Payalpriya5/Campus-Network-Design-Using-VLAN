# Campus-Network-Design-Using-VLAN
A campus network design project using VLANs in Cisco Packet Tracer, implementing multi-department segmentation, inter-VLAN routing, and DHCP configuration.
This project demonstrates the design and implementation of a Campus Network using VLANs in Cisco Packet Tracer.

The network is divided into multiple departments such as Admin, Faculty, and Students using VLAN segmentation. Inter-VLAN communication is achieved using a Layer 3 switch, and DHCP is configured for automatic IP address allocation.

Key Features:
- VLAN segmentation for network isolation
- Inter-VLAN routing using Layer 3 switch
- DHCP configuration for automatic IP assignment
- Hierarchical network design (Core, Distribution, Access)
- Scalable and efficient network architecture

Tools Used:
- Cisco Packet Tracer
- Networking Concepts (VLAN, Routing, Switching)
